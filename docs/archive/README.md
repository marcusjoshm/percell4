# docs/archive/

Historical documents that have been **superseded** by current code or current
docs. Files here are read-only reference for "what we used to think" — they
are not maintained and must never be treated as a source of truth for how the
app works today.

Every file in this directory must start with a `> **SUPERSEDED — YYYY-MM-DD.**`
banner explaining what replaced it and where to find the current canonical
description.

If you need to know how PerCell4 works *now*, read `CLAUDE.md` at the repo
root and the per-module `CLAUDE.md` files under `src/percell4/`.
